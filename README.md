# FlySky Network - Telegram Mini App

Earn FSN tokens by mining virtual flights and watching ads.

## Live Demo
- Web: https://mini.fsncrew.io
- Telegram: @fsncrewbot

## Features
- Flight Mining System
- Rewarded Ads System  
- Package Upgrades
- Referral System
- Telegram WebApp Integration

## Mining Packages
- Economy: 10 FSN every 12 hours, 15 ads/day
- Business: 50 FSN every 12 hours, 100 ads/day
- First: 100 FSN every 12 hours, 500 ads/day

## Quick Start
1. Open in Telegram via @fsncrewbot
2. Login with Telegram account
3. Start mining immediately

## Deployment
This app is deployed on Vercel at https://mini.fsncrew.io

## Tech Stack
- HTML5, CSS3, JavaScript
- Telegram WebApp SDK
- Vercel for hosting
