# Telegram Bot Setup Guide

## Steps to connect your Mini App to Telegram:

1. **Create/Configure Bot via @BotFather**
   - Open @BotFather in Telegram
   - Use /newbot to create new bot or /mybots to edit existing
   - Bot name: FlySky Network
   - Username: fsncrewbot

2. **Set Menu Button**
   - In @BotFather, select your bot
   - Go to Bot Settings > Menu Button
   - Configure menu button:
     - URL: https://mini.fsncrew.io
     - Text: 🚀 Open FlySky

3. **Test the Integration**
   - Open Telegram
   - Search for @fsncrewbot
   - Click the menu button
   - Mini App should open fullscreen

4. **Troubleshooting**
   - Ensure Vercel deployment is live
   - Check URL is HTTPS
   - Clear Telegram cache if needed
   - Test in browser first
